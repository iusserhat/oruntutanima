🌳 Decision Tree (Karar Ağacı) Nedir?

Decision Tree, ağaç yapısında sorular sorarak veriyi sınıflandıran veya regresyon yapan bir algoritmdır. Her düğümde bir özelliğe bakılır ve soru sorulur. Yanıt sizi bir alt dala götürür. En sonunda yaprak düğüme ulaşıldığında tahmin verilir.

Analoji: "Dışarıya çıkayım mı?" sorusunu kendine soruyorsun. "Hava güneşli mi?" → Evet → "Yorgun muyum?" → Hayır → "Evet, çık!". Bu tam olarak bir Decision Tree mantığı!
📌
Nottan Birebir
"Decision Tree: Veriden alınan eğitim bilgisi ile bir iç yapı (tree) oluşturulur. Bu yapı üzerinden veri kümesi daha küçük boyutlara bölünür."

⚙️ Hunt's Algoritması

Decision Tree'lerin temel inşa algoritmasıdır. Özyinelemeli (recursive) çalışır:

  1. Düğümdeki tüm örnek aynı sınıfsa → YAPRAK yap
    Pure node bulundu. Ağacın bu dalı tamamlandı. Artık bölme yok.
  2. Farklı sınıflar varsa → En iyi özelliği bul ve böl
    Information Gain hesaplanır. En yüksek IG'li özellik seçilip veri bölünür.
  3. Her alt düğüm için 1. ve 2. adımı tekrarla
    Özyinelemeli devam eder. Tüm yapraklar pure olana dek sürer.
📌
Nottan Birebir
"Düğüm oluşturulur; Eğer tüm örnek aynı sınıfa aitse, düğüm yaprak düğümdür ve sınıf etiketi bir değer atanır; aksi takdirde tüm özellikler için veri kümesi alt gruplara bölündürülür; özellik seçmek için bilgi kazancı (info gain) kullanılır."

🔬 Entropy (Entropi) Nedir?

Entropy, bir düğümdeki bozukluk (impurity / belirsizlik) ölçüsüdür. Düşük entropy → düğüm saf (tek sınıf). Yüksek entropy → düğüm karışık (birden fazla sınıf).

Analoji: Bir çantadan kırmızı bilye çekmek istiyorsun. Çantada sadece kırmızı blyeler varsa → Kesin kırmızı çekersin → belirsizlik=0 → Entropy=0. Çantada 50 kırmızı 50 mavi varsa → Eşit şans → Maximum belirsizlik → Entropy=1 (maksimum).

📐 Entropy Formülü

Entropy(t) = − Σ [P(i|t) · log₂(P(i|t))]

Burada:
t = Düğüm (node)
i = Sınıf (class)
P(i|t) = Düğüm t'de sınıf i'nin oranı (kaçta kaç)
log₂ = 2 tabanında logaritma
🚨
Kritik Kurallar — Ezber
  • Tüm örnek aynı sınıftan → P=1 → log₂(1)=0 → Entropy = 0 (SAFI/PURE)
  • İki sınıf eşit sayıda → P=0.5 → Entropy = 1 (MAKSIMUM)
  • Entropy her zaman 0 ile 1 arasında (ikili sınıflama için)
  • log₂(0) tanımsızdır → 0·log₂(0) = 0 kabul edilir
Özel değerler:
log₂(1) = 0   |   log₂(2) = 1   |   log₂(4) = 2   |   log₂(0.5) = -1   |   log₂(0.25) = -2

📊 Information Gain (Bilgi Kazancı)

Bir özelliğe göre bölme yapıldığında ne kadar "kirlilik (entropy)" azalıyor? Information Gain bunu ölçer. En yüksek IG'li özellik seçilir!

Info_Gain(A) = Entropy(Parent) − Σ [(|Dᵥ| / |D|) · Entropy(Dᵥ)]

Parent = Mevcut düğüm
A = Test edilecek özellik
Dᵥ = Özellik A'nın v değerine sahip örnekler
|Dᵥ|/|D| = Alt setin ağırlığı (toplam içindeki oranı)
💡
Kolay Hatırlatma
IG = Önce tüm düğümün Entropy'si − Sonra (her alt düğümün ağırlıklı Entropy'si)
IG ne kadar büyükse, o özellik o kadar faydalıdır → Root Node olma adayı odur!

🔢 Adım Adım Tam Hesaplama

🚨
Sınav: Bu Hesaplamayı Adım Adım Yap!

📍 ADIM 1 — Başlangıç Entropy'si (Kök Düğüm)

14 örnekte: 9 YES, 5 NO

P(Yes) = 9/14 = 0.643
P(No) = 5/14 = 0.357

Entropy(Root) = −P(Yes)·log₂(P(Yes)) − P(No)·log₂(P(No))
= −(9/14)·log₂(9/14) − (5/14)·log₂(5/14)
= −0.643·(−0.637) − 0.357·(−1.485)
= 0.410 + 0.530
Entropy(Root) = 0.940

📍 ADIM 2 — Outlook ile IG Hesabı

Outlook 3 değer alıyor: Sunny (5 örnek), Overcast (4 örnek), Rain (5 örnek)

Sunny için Entropy (2 Yes, 3 No):

E(Sunny) = −(2/5)·log₂(2/5) − (3/5)·log₂(3/5)
= −0.4·(−1.322) − 0.6·(−0.737)
= 0.529 + 0.442 = 0.971

Overcast için Entropy (4 Yes, 0 No):

E(Overcast) = −(4/4)·log₂(4/4) − 0
= −1·log₂(1) = −1·0
E(Overcast) = 0.000 ← PURE NODE! (Tüm Yes)

Rain için Entropy (3 Yes, 2 No):

E(Rain) = −(3/5)·log₂(3/5) − (2/5)·log₂(2/5)
= −0.6·(−0.737) − 0.4·(−1.322)
= 0.442 + 0.529 = 0.971

Outlook için Bilgi Kazancı:

IG(Outlook) = E(Root) − [ (5/14)·E(Sunny) + (4/14)·E(Overcast) + (5/14)·E(Rain) ] = 0.940 − [ (5/14)·0.971 + (4/14)·0.000 + (5/14)·0.971 ] = 0.940 − [ 0.347 + 0.000 + 0.347 ] = 0.940 − 0.694 IG(Outlook) = 0.246

📐 Gini Index (Alternatif Saflık Ölçüsü)

Entropy yerine kullanılan başka bir bozukluk ölçüsüdür. sklearn'ın varsayılan kriteri Gini'dir.

Gini(t) = 1 − Σ [P(i|t)²]

İkili sınıf için: Gini = 1 − [P(Yes)² + P(No)²]

Pure node: Gini = 0      Maksimum (50/50): Gini = 0.5

Entropy vs Gini

  • İkisi de aynı sonuçları verir genellikle
  • Entropy: log₂ hesabı gerekir (yavaş)
  • Gini: sadece kare işlemi (daha hızlı)
  • sklearn default: Gini
  • Notlarda Entropy daha fazla gösterilmiş

Gini Örnek Hesap

14 örnekte 9 Yes, 5 No:

Gini = 1 − [(9/14)² + (5/14)²]
= 1 − [0.413 + 0.127]
= 1 − 0.540 = 0.460

✂️ Overfitting ve Pruning

Decision Tree çok derine inerse overfitting yapar: Eğitim verisini ezberler, yeni veriye genelleme yapamaz.

Overfitting Belirtileri

  • Eğitim accuracy çok yüksek
  • Test accuracy çok düşük
  • Ağaç çok derin ve karmaşık

Pruning Çözümü

  • max_depth: Ağaç derinliği sınırlandırılır
  • min_samples_split: Bölüm için min örnek
  • min_samples_leaf: Yaprak için min örnek

💻 Kod Örnekleri

from sklearn.tree import DecisionTreeClassifier, export_graphviz
from sklearn import preprocessing, metrics
import pandas as pd, numpy as np

# ── Veri Hazırlık ──
df = pd.read_csv('drug200.csv')
X = df.drop('Drug', axis=1)
y = df['Drug']

# ── LabelEncoder (Sınavda çıkabilir!) ──
le = preprocessing.LabelEncoder()
X['Sex']        = le.fit_transform(X['Sex'])        # M/F → 0/1
X['BP']         = le.fit_transform(X['BP'])         # LOW/NORMAL/HIGH → 0/1/2
X['Cholesterol'] = le.fit_transform(X['Cholesterol']) # NORMAL/HIGH → 0/1

# ── Train/Test ──
from sklearn.model_selection import train_test_split
X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.3, random_state=3
)

# ── Önemli! Decision tree scikit-learn ──
drugTree = DecisionTreeClassifier(
    criterion='entropy',  # Entropy kullan (Gini de olabilir)
    max_depth=4            # Overfitting önlemi
)
drugTree.fit(X_train, y_train)

predTree = drugTree.predict(X_test)
print("Accuracy:", metrics.accuracy_score(y_test, predTree))

# ── Görselleştirme ──
import pydotplus
dot_data = export_graphviz(
    drugTree,
    feature_names=['Age', 'Sex', 'BP', 'Cholesterol', 'Na_to_K'],
    class_names=drugTree.classes_,
    filled=True, special_characters=True
)
graph = pydotplus.graph_from_dot_data(dot_data.getvalue())
graph.write_png('dtree.png')   # Ağaç görüntüsü kaydedilir
