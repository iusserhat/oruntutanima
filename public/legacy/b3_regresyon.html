📊 Regresyon Nedir?

Regresyon, sürekli (continuous) bir çıktıyı tahmin etmek için kullanılan Supervised Learning yöntemidir. Tahmin edilmek istenen değer sayısal ve süreklidir (net bir değer, örneğin: 234 g/km CO₂).

Analoji: Arabanın motor hacmini biliyorsun (3.5 litre). Bunun CO₂ salınımının kaç g/km olacağını tahmin etmek istiyorsun. Elinde binlerce arabanın motor hacmi ve CO₂ verisi var. Regresyon işte bunu öğrenir.
📌
Temel Kural
Bağımlı değişken (y) sürekli ise → Regresyon | ayrık ise → Classification.
Regresyonda bağımlı değişken kesinlikle sürekli (continuous) olmalıdır.

📏 Simple Linear Regression (SLR)

Sadece 1 bağımsız değişken kullanılır. Scatter plot'ta noktalara en yakın düz çizgi çizilir (en az MSE'li çizgi seçilir).

ŷ = θ₀ + θ₁·x₁
θ₀ (theta-0)
Başlangıç noktası (intercept) — çizginin y eksenini kestiği nokta
θ₁ (theta-1)
Feature'ın katsayısı (coefficient) — çizginin eğimi
x₁
Seçilen bağımsız değişken (feature), örneğin enginesize
ŷ (y-hat)
Tahmin edilen bağımlı değişken — gerçek değer değil, tahmin!
📌
Feature Seçimi — Scatter Plot ile
10 bağımsız değişkenden 1 tanesi seçilecekse, bağımlı değişken (CO₂) ile en yüksek korelasyona sahip olan seçilir. plt.scatter(feature, CO2E) ile görselleştir → düz çizgiye en yakın olanı seç.

📉 Scatter Plot & MSE (Model Değerlendirme)

Model elde edildikten sonra başarım ve hata oranı ölçülmeli.

MetrikNe Ölçer?İyi Olan

⚠️
Train vs Test Accuracy Farkı
Eğer train ve test accuracy arasında 6 puandan fazla fark varsa → Overfitting!
Test verileri gerçek verilerden oluşmalıdır (sentetik/augmentli olmamalı — ama train'de kullanılabilir).
📌
Karşılaştırma Örneği
SLR: MAE = 22.71, R²-score = 0.77
Polynomial: MAE = 25.28, R²-score = 0.75
→ SLR daha iyi! (MAE küçük, R² büyük)

🔢 θ₁ ve θ₀ Formülleri — Manuel Hesaplama

θ₁ = Σ(xᵢ − x̄)·(yᵢ − ȳ) / Σ(xᵢ − x̄)²

θ₀ = ȳ − θ₁ · x̄
  • x̄ → Seçilen bağımsız değişkene ait sütunun ortalaması
  • ȳ → Bağımlı değişkene ait sütunun ortalaması
  • xᵢ, yᵢ → Her satırdaki değerler (Her satır ayrı ayrı hesaplanır)
📌
Kod ile CO₂ Veri Seti Sonucu
θ₁ = 39.093 (coefficient), θ₀ = 125.38 (intercept)
Model: ŷ = 125.38 + 39.093 × (enginesize)

Yeni değer: enginesize = 3.5 → ŷ = 125.38 + 39.093×3.5 = 262.2 g/km CO₂

📐 Multiple Linear Regression (MLR)

Birden fazla bağımsız değişken kullanılır. Çok bağımsız değişken olduğundan matris şeklinde yazılır.

ŷ = θ₀ + θ₁x₁ + θ₂x₂ + θ₃x₃ + ... + θₙxₙ

Matris gösterimi: ŷ = θᵀX
θᵀ = [θ₀, θ₁, θ₂, θ₃...] , X = [x₀, x₁, x₂, x₃]ᵀ
📌
CO₂ Veri Seti Örneği (3 bağımsız değişken)
ŷ = 62.91 + 9.17·(enginesize) + 8.40·(cylinders) + 9.84·(fuel_consumption)
variance score = 0.86 → 3 feature ile daha iyi başarım elde edildi.

4 feature seçildi (3 bağımsız, 1 bağımlı) → 3 katsayı (coefficient) bulunur.
Bu değerler → array'e dönüştürülür → Coefficients (katsayılar) bulunur.
⚠️
MLR Kritik Notlar
Regresyonda bağımlı değişken kesinlikle sürekli (continuous) olmalı! θᵀ alınamayacağından θ'nın transpozu alınır (çünkü ŷ = θᵀ bu şekilde sütun vektörüyle çarp). SLR ne zaman tercih edilir? Sadece 1 bağımsız değişken olduğunda, ya da 1 feature bağımlıyla güçlü korelasyona sahipse.

🌀 Polynomial Regresyon

Non-linear'ın alt kümesidir. Sadece 1 değişken ve bu değişken pozitif derecelerde kullanılır.

Derece 2: ŷ = θ₀ + θ₁x + θ₂x²
Derece 3: ŷ = θ₀ + θ₁x + θ₂x² + θ₃x³
📌
CO₂ Veri Seti Örneği (degree=2)
CO₂Em = 101.73 + 0.54·(ES) − 2.01·(ES)²
Karşılaştırma: SLR R²=0.77 > Polynomial R²=0.75 → SLR bu durumda daha iyi.

Polynomial Regresyon Adımları

  1. Derece (D=2, D=3...) belirle — veriyi görselleştirerek karar ver
  2. PolynomialFeatures ile veriyi matrise dönüştür
  3. LinearRegression ile modeli eğit
  4. Coefficients ve Intercept'i al → ŷ formülünü yaz
from sklearn.preprocessing import PolynomialFeatures
from sklearn.linear_model import LinearRegression

poly = PolynomialFeatures(degree=2)
train_x_poly = poly.fit_transform(train_x)  # train_x → matrise çevrildi

clf = LinearRegression()
clf.fit(train_x_poly, train_y)

print("Coefficients:", clf.coef_)   # θ₁, θ₂ değerleri
print("Intercept:", clf.intercept_)   # θ₀ değeri
⚠️
Önemli Not
train_x yerine train_x_poly kullanılır — çünkü veriler artık matris halinde. Matris dönüşümü örneği: [2, 3, 1] → derece 2 → [[1,2,4], [1,3,9], [1,1,1]]

🔄 Polynomial mu, Non-linear mi? — Nasıl Ayırt Edilir?

ÖzellikPolynomialNon-linear (Daha Geniş)
Değişken sayısıHer zaman 1 değişkenPolynomial'ı kapsar, 1+ değişken
Derece türüPozitif tam sayı (x¹, x², x³...)eˣ, ln(x), x⁻², sin(x) gibi her türlü
Örnek denklemy = x² + 3x + 1y = eˣ, y = log(x), y = 1/(1+eˣ)
2 değişken var → MLR (polynomial değil, non-linear değil)
1 değişken, pozitif derecelerde → Polynomial
1 değişken, eˣ veya log gibi → Non-linear

🌊 Non-Linear Regresyon

Polynomial bile yetmezse kullanılır. Polynomial'ı kapsar ama daha geniştir.

x = np.arange(-5, 5, 0.1)  # -5'ten 5'e kadar, 0.1 aralıklı
y = 2*x + 3
y_noise = 2 * np.random.normal(size=x.size)  # Noise ekleniyor (gerçek veri simülasyonu)
ydata = y + y_noise
plt.plot(x, ydata)
📌
Model Seçimi Nasıl Yapılır?
2 model: MSE ve R² karşılaştırılır. MSE küçük + R² büyük olan seçilir. Veri düzleyse → SLR/MLR | S şeklindeyse → Sigmoid | U şeklindeyse → Quadratic

⚯ Sigmoidal Non-linear Regresyon (GDP Örneği)

🚨
SINAV — Sigmoid formülü bilinmeli! Büyük ihtimalle çıkar.
Veri S-şeklinde bir eğri izliyorsa → Sigmoidal. GDP (Gayri Safi Hasıla) grafiği sonsuz artamayacağından Sigmoid kullanılır.
y = 1 / (1 + e^(−β₁(x−β₂)))
  • β₁ → Eğrinin eğimini (ne kadar hızlı geçiş) belirler
  • β₂ → Eğrinin merkezini belirler (geçiş noktası — inflection point)

Sigmoid Python Kodu

from scipy.optimize import curve_fit

def sigmoid(x, Beta_1, Beta_2):
    y = 1 / (1 + np.exp(-Beta_1 * (x - Beta_2)))
    return y

# Başlangıç değerleri (ilk tahmini girdiler)
beta_1 = 0.3
beta_2 = 1990

# curve_fit ile en iyi β₁ ve β₂ değerleri bulunur
# *popt → iterable olarak açılır (best_beta1, best_beta2)
popt, pcov = curve_fit(sigmoid, xdata, ydata)

# Tahmin — popt en iyi parametreler
y_pred = sigmoid(xdata, *popt)

# Modelin accuracy'si de hesaplanabilir
print("R²:", r2_score(ydata, y_pred))
📌
Nottan — Sigmoid Detayları

Model ile gerçek değerler uyuşmadığında → scipy'den curve_fit kullanılarak en optimum β₁ ve β₂ bulunur.

Ayrıca veri normalize edilir (0-1 arasına): 1, 2, 3, 30, 40 → 1/40, 2/40, 3/40, 30/40, 40/40

popt neden *popt şeklinde? popt = [best_beta1, best_beta2] listesi → *popt ile ayrı parametrelere açılır.

Bu data ile daha yakın tahmin yapan model elde edilir. Modelin accuracy'si hesaplanabilir.

💻 Kod Özeti

import pandas as pd
import numpy as np
from sklearn.linear_model import LinearRegression

# Veri oku ve feature seç
df = pd.read_csv("FuelConsumption.csv")
cdf = df[['ENGINESIZE', 'CYLINDERS', 'FUELCONSUMPTION_COMB', 'CO2EMISSIONS']]

# Train/test ayır (%80 train, %20 test)
msk = np.random.rand(len(df)) < 0.8
train = cdf[msk]
test = cdf[~msk]

# SLR — 1 satırda eğitim!
clf = LinearRegression()
clf.fit(train[['ENGINESIZE']], train['CO2EMISSIONS'])

print("θ₁ (Coefficient):", clf.coef_)
print("θ₀ (Intercept):", clf.intercept_)

# Tahmin
yhat = clf.predict(test[['ENGINESIZE']])

# Değerlendirme
print("MAE:", np.mean(np.absolute(yhat - test['CO2EMISSIONS'])))
print("R²:", clf.score(test[['ENGINESIZE']], test['CO2EMISSIONS']))

Bölüm 3 Sınav Özeti
  • SLR: ŷ = θ₀ + θ₁·x | θ₁ = Σ(xᵢ−x̄)(yᵢ−ȳ) / Σ(xᵢ−x̄)²
  • R² büyük, MSE küçük → Bu model daha iyi
  • Sigmoid: y = 1/(1+e^(−β₁(x−β₂))) → GDP örneği → S-şekli → sigmoidal
  • Polynomial: 1 değişken, pozitif dereceleri → PolynomialFeatures(degree=2)
  • linear ⊂ polynomial ⊂ non-linear
  • Train/test farkı >6 puan → Overfitting
  • variance score = 0.86 (MLR) → 3 feature → daha iyi başarım
  • Grafik şekline göre model seç tablosunu ezberle
