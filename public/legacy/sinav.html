📐 Formüller — Hepsini Bil

SLR: ŷ = θ₀ + θ₁·x
MLR: ŷ = θ₀ + Σθᵢxᵢ = θᵀX
Sigmoid: y = 1/(1+e^(−β₁(x−β₂)))
Z-score: Z = (x − μ) / σ
Min-Max: x' = (x−min)/(max−min)
IQR = Q3 − Q1 | Alt=Q1−1.5·IQR | Üst=Q3+1.5·IQR
Entropy = −Σ P(i|t)·log₂P(i|t)
IG = E(parent) − Σ(|Dᵥ|/|D|)·E(Dᵥ)
Precision = TP/(TP+FP)
Recall = TP/(TP+FN)
F1 = 2·P·R/(P+R)
Wavelet a = (x₁+x₂)/2 | d = (x₁−x₂)/2

🔴 Entropy Hesaplama Adımları

  1. P(Evet) = k/n, P(Hayır) = (n−k)/n
  2. E = −P(E)·log₂P(E) − P(H)·log₂P(H)
  3. Her özellik için E hesapla
  4. IG = E(kök) − Σ (ağırlık×E(alt))
  5. En yüksek IG → Root Node
Tenis: E(14:9Y,5N) = −(9/14)·log₂(9/14) − (5/14)·log₂(5/14) ≈ 0.94
E(Overcast) = 0.000 (PURE!)
Gain(D,Outlook) = 0.424 → EN YÜKSEK → ROOT!
Gain(D,Wind) = 0.102 | Gain(D,Humidity) = 0.016 | Gain(D,Temp) = 0.008

🟣 IQR Adım Adım

Veri: {4, 5, 7, 8, 9, 10, 12, 13, 15, 30} (sıralı)

  • Q2 (medyan) = (9+10)/2 = 9.5
  • Q1 (alt yarı medyanı) = 7
  • Q3 (üst yarı medyanı) = 13
  • IQR = Q3−Q1 = 13−7 = 6
  • Alt = Q1−1.5×IQR = 7−9 = −2
  • Üst = Q3+1.5×IQR = 13+9 = 22
  • 30 > 22 → 30 OUTLIER!

🟣 Grafik → Hangi Model?

  • Düz çizgi ↗ → Linear: y=ax+b
  • Yükselen eğri ↗ → Polynomial: y=ax²+bx+c
  • U şekli ∪ → Quadratic: y=x²
  • S şekli ⌇ → Sigmoid: y=1/(1+e^(−β₁(x−β₂)))
  • Üstel büyüme ↑↑ → Exponential: y=a+b·cˣ
  • Yavaşlayan artış → Logarithmic: y=log(x)

🟢 Hangi Metrik Ne Zaman?

  • Regresyon → MSE, MAE, R²
  • Dengeli veri → Accuracy
  • Dengesiz veri → F1 Score
  • Yanlış alarmı azalt → Precision
  • Hiçbir pozitifi kaçırma → Recall
  • Olasılık bazlı → LogLoss
FN = Gerçek Pozitif ama Negatif dedik → EN TEHLİKELİ FP = Gerçek Negatif ama Pozitif dedik → Yanlış Alarm

🟢 Wavelet Formülleri

a = (x₁ + x₂) / 2 → Ortalama (taşınır)
d = (x₁ − x₂) / 2 → Fark (saklanır)

8 elemanlı veri → 3 adımda → 1 özet değer
16 elemanlı veri → 4 adımda → 1 özet değer

Sıkıştırma: 0 olanlar kaldırılır VEYA küçük (threshold) değerler atılır.

Örnek (16 değer sonucu):
A₄ = {14.25} ← Tek özet değer!
D değerleri: saklanır veya küçükleri atılır.

🟢 Kavramlar — Hızlı Hatırlama
